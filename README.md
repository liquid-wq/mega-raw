<a name="deutsch"></a>

**RetroAchievements auf echter Sega Mega Drive / Genesis Hardware — über das Mega EverDrive.**

Dieses Projekt gibt es in zwei Versionen:

## 🟢 [cpp/](cpp/) — empfohlen

Fertige `.exe` für Windows, kein Python nötig. Einfach entpacken und starten.
→ [C++-README lesen](cpp/README.md)

**Zum Spielen brauchst du nur den Ordner `cpp/Release/`** — lade dir nur
diesen herunter, dort liegt die fertige `.exe`. Der Ordner `cpp/src/` enthält
den Quellcode zur Transparenz (damit jeder selbst nachvollziehen kann, dass
keine Schadsoftware enthalten ist — nicht zur freien Weiterverwendung) und
wird zum Spielen nicht benoetigt.

## 🐍 [python/](python/)

Die ursprüngliche Referenzversion als lesbare Python-Dateien, kein `.exe`.
Für alle, die den Quellcode direkt einsehen und ausführen möchten.
→ [Python-README lesen](python/README.md)

---

Beide Versionen sind funktional identisch, aber die C++-Version ist die
kompatiblere Wahl (kein Python nötig, weniger Antiviren-Fehlalarme) — die
Python-Version bleibt aus Transparenzgründen parallel verfügbar.

## Probleme melden

Dies ist ein Hobby-Projekt — bei tausenden Mega-Drive-Spielen ist es kaum
machbar, alle einzeln zu testen. Wenn ein Spiel nicht startet, ein Achievement
falsch triggert, oder sonst etwas nicht funktioniert: bitte über den
[Issues-Tab](../../issues) melden, am besten mit Spielname, ROM-Region und
kurzer Beschreibung was passiert ist. Jede Meldung hilft, die Abdeckung
schrittweise zu verbessern.

<a name="english"></a>

**English:**

This project comes in two versions:

## 🟢 [cpp/](cpp/) — recommended

Ready-to-run `.exe` for Windows, no Python required. Just unpack and run.
→ [Read the C++ README](cpp/README.md)

**To play, you only need the `cpp/Release/` folder** — download just that
one, the ready-to-run `.exe` is in there. The `cpp/src/` folder contains the
source code for transparency (so anyone can see for themselves that there's
no malware inside — not for free reuse) and isn't needed to play.

## 🐍 [python/](python/)

The original reference version as readable Python files, no `.exe`.
For anyone who wants to inspect and run the source code directly.
→ [Read the Python README](python/README.md)

---

Both versions are functionally identical, but the C++ version is the more
compatible choice (no Python required, fewer antivirus false positives) —
the Python version stays available in parallel for transparency.

## Reporting issues

This is a hobby project — with thousands of Mega Drive games out there, it's
practically impossible to test every single one. If a game doesn't boot, an
achievement triggers incorrectly, or anything else is off: please report it
via the [Issues tab](../../issues), ideally with the game name, ROM region,
and a short description of what happened. Every report helps improve
coverage step by step.
